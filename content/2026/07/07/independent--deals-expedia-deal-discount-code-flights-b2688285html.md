## How to use an Expedia discount code

Once you’ve clicked the "see code" button on the offer, copy the code to save for later. When you are ready to proceed with your booking on the Expedia website, find the "price details" widget. Look for the "use a coupon, credit or promotion code" link, and click it to reveal a box. Paste the code directly and click apply to add the discount to your total.

## Why isn't my Expedia discount code working?

Many Expedia voucher codes only apply to specific hotels, destinations or booking types, so it's worth checking the terms before you pay. During our latest testing, we found the most common reasons a code wouldn't apply were that the hotel was already discounted, the booking used the pay-later option or the property wasn't participating in the promotion.

When trying to apply a discount code, be sure to check the following:

- Your booking meets minimum spend requirements
- The hotel or holiday is included in the promotion
- You're paying at the time of booking, if required
- You're entering the code exactly as shown
- The offer hasn't expired

If your code still doesn't work, compare the price with One Key member rates or the Expedia app, as these often offer similar or even bigger savings without needing a voucher code.
